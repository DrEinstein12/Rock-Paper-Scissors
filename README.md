Rock Paper Scissors (ATmega32U4 Assembly)

A low-level implementation of the classic Rock–Paper–Scissors game using AVR Assembly on the ATmega32U4 microcontroller.
This project demonstrates embedded systems concepts including USART communication and external interrupts.

Overview:

This project implements a two-player (or player vs system) Rock–Paper–Scissors game using microcontrollers. 
Communication between devices (or with a terminal) is handled via USART, while external interrupts are used for responsive user input.

Objectives:

Practice AVR Assembly programming
Use USART for serial communication
Implement external interrupts for real-time input handling
Apply embedded systems design to a simple interactive game
